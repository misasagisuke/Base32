Base32.js
====

Encoding and decoding BASE32 in JavaScript.

## Usage
 Encode
 ```javascript
 var passphrase = 'hogehoge';
 var base32_str = encobase32_encodede(passphrase);
 ```
 Decode
 ```javascript
 var passphrase = base32_decode(base32_str);
```
